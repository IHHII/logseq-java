- 编程语言发展
	- 1. [[二进制]]
		- 第一代：二进制编程语言/机器语言
		- 组成：0和1
		- 特点：阅读性极差，维护性极差,开发效率非常低下
		- ```
		  00010011
		  ```
	- 2. [[汇编语言]]（符号语言）
		- 思想：使用一些特殊的符号，替代难于理解的01
		- AB-10101111
		  &A-00110101
		- ```
		  AB &A CB
		  ```
		- 特点：阅读性，维护性仍旧很差，但是简化了大量的01组合
	- 3.[[高级编程语言]]
		- 采用人类的语言作为编程的标准
		- ```java
		  system.out.println("Hello");
		  ```

			- 结构化编程[[面向过程]]
				- C语言
			- [[面向对象]]
				- C++
				- Java

				- C#
